## MCP Server

Install dependencies with Poetry:

```shell
poetry install
```
To run MCP server:

```shell
python mcp_server.py
```

## MCP inspector

Install MCP inspector:

```shell
npm install -g @modelcontextprotocol/inspector
```

Run MCP inspector:

```shell
npx @modelcontextprotocol/inspector
```
